# Anter: 지능형 SSH 클라이언트

Anter는 개발자와 시스템 관리자를 위해 설계된 현대적인 크로스 플랫폼 SSH 클라이언트입니다. Flutter로 제작되어 아름다운 UI와 함께 세션 녹화, AI 명령어 보조, 스마트 백업 등 강력한 기능을 제공합니다.

![Anter Logo](assets/images/anter_logo.png)

## 주요 기능 🚀

### 1. 현대적이고 반응성 높은 터미널

- **크로스 플랫폼:** macOS, Windows, Linux, Android, iOS에서 완벽하게 동작합니다.
- **아름다운 UI:** 다크 모드와 커스터마이징 가능한 테마를 지원하는 직관적인 인터페이스.
- **탭 관리:** 브라우저와 유사한 탭 시스템으로 여러 SSH 세션을 효율적으로 관리하세요.
- **분할 화면:** 멀티태스킹을 위한 터미널 분할 뷰를 지원합니다.
- **가상 키 툴바:** (모바일) Android/iOS 환경에서 Esc, Tab, 방향키, Ctrl 조합 등을 빠르게 입력할 수 있는 전용 툴바를 제공합니다.

### 2. 세션 녹화 및 재생 🎥

- **자동 녹화:** 감사 및 검토를 위해 모든 터미널 세션을 자동으로 녹화할 수 있습니다.
- **다시보기:** 내장된 플레이어를 통해 과거 세션을 타임라인과 함께 생생하게 다시 볼 수 있습니다.
- **개별 내보내기:** 필요한 녹화 파일만 별도로 내보내어 공유하거나 보관할 수 있습니다.

### 3. AI 어시스턴트 (Gemini) 🤖

- **스마트 명령어 생성:** Google Gemini AI가 통합되어 자연어 질문을 리눅스 명령어로 변환해 줍니다.
- **문맥 인식:** "디스크 용량 확인해줘" 또는 "큰 파일 찾아줘"와 같이 물어보면 실행할 정확한 명령어를 제안합니다.
- **안전한 실행:** 생성된 명령어는 사용자가 검토한 후 실행할 수 있어 안전합니다.

### 4. 백업 및 복구 ☁️

- **백업:** 세션 정보, 앱 설정, 단축키 설정을 JSON 파일 하나로 가볍게 백업합니다.
- **복구:** 중복 세션 생성을 방지하는 지능형 복구 로직을 갖췄습니다.
- **기기 간 이동:** PC와 모바일 간에 설정을 자유롭게 공유할 수 있습니다.

### 5. 고급 SSH 기능 🔐

- **키 인증:** PEM, PPK, OpenSSH 키 형식을 지원합니다.
- **로그인 스크립트:** 로그인 후 자동으로 수행할 작업을 스크립트로 정의하여 자동화할 수 있습니다.
- **SFTP 지원:** 간편한 파일 전송을 위한 SFTP 클라이언트가 내장되어 있습니다.
- **스마트 터널링:** 포트 포워딩 터널을 쉽고 빠르게 설정할 수 있습니다.
- **프록시 점프 (Gateway):** Bastion 호스트나 게이트웨이를 경유하여 내부 서버에 안전하게 접속할 수 있습니다.
- **키워드 알림:** 터미널 출력에서 특정 키워드(예: "error", "done")가 감지되면 모바일 푸시 알림을 발송합니다.

## 시작하기

### 필수 조건

- Flutter SDK (최신 안정 버전)
- Dart SDK

### 설치 방법

1.  **저장소 복제 (Clone)**

    ```bash
    git clone https://github.com/parkjw/anter.git
    cd anter
    ```

2.  **의존성 설치**

    ```bash
    flutter pub get
    ```

3.  **앱 실행**

    ```bash
    # macOS
    flutter run -d macos

    # Android
    flutter run -d android
    ```

## 아키텍처

Anter는 **Riverpod** 상태 관리와 **Drift** 로컬 데이터베이스를 사용하여 클린 아키텍처 원칙을 따릅니다.

- `lib/src/features`: 기능별 모듈 (세션, 터미널, 설정 등)
- `lib/src/core`: 코어 유틸리티, 데이터베이스 설정, 공용 위젯
- `lib/src/shared`: 공유 UI 컴포넌트 및 헬퍼
